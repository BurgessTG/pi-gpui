# Provider logo sources

These SVGs cover every built-in `@earendil-works/pi-ai` provider slug used by the desktop auth UI.

Simple Icons is the source of truth where the brand exists there; those assets use the Simple Icons SVG path plus its `hex` brand color. For providers that Simple Icons does not currently publish, the asset comes from the closest first-party/official brand source. Regional/product-plan slugs intentionally share the parent brand asset where the SDK provider is the same company.
